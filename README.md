# Excel_Retail Analysis Project
This project analyzes the sales data from a fashion store to gain insights into their sales performance, products performance and trends, then finally, the regional breakdown. The analysis was performed using Excel.

## Dataset Overview
The dataset consists of the sales records from 2023 to 2024, containing 9,922 rows and 12 columns in total(7 of the columns makes up the original data, the remaining 5 columns were added; The order date was splitted to get the 'Year', 'Month', 'Quarter', and the 'Day' The last column was to get the Total revenue generated by multiplying the unit price with the quantity sold. Below is a brief description of each column:

| Column Name     | Description                                    | Data Type |
| --------------- | ---------------------------------------------- | --------- |
| OrderID         | Unique identifier for each order               | Integer   |
| Customer Id     | Unique identifier for each customer            | Text      |
| Product         | Name of the product sold                       | Text      |
| Region          | Region of the customer                         | Text      |
| OrderDate       | Date when the order was placed                 | Date      |
| Quantity        | Number of product sold                         | Integer   |
| Unit Price      | Selling price of the product sold              | Integer   |
| Year            | The year the product was sold                  | Text      |
| Month Name      | The month the product was sold                 | Text      |
| Quarter         | The quarter of the year the product was sold   | Integer   |
| Day Name        | The day of the week the product was sold       | Text      |
| TotalSales      | Total sales value of the product (in USD)      | Float     |

## Data Preparation and Processing
- Splitted the order date to 4 parts: 'Year', 'Month_Name', 'Quarter' and 'Day_Name. 
- Cleaned the data by removing duplicates. 
- Added a new column for total sales based on the formula: `Total_Sales = Unit Price * Qunatity`.
- Grouped sales by quantity sold, total profit made by each region using a pivot table for easier analysis.

By examining customer behaviors, subscription types, and churn rates, this project provides valuable insights for stakeholders to improve customer retention, optimize marketing, and drive growth.

## Files and Repository Structure

### 1. Introduction
The project explores various customer segments to understand subscription trends and churn behavior. Using Excel pivot tables, functions, and charts, the analysis offers an in-depth view of customer behavior patterns that could aid in decision-making.

### 2. Project Objectives
   a) Primary Goal: To identify key factors affecting customer subscriptions and cancellations.
   
   b) Secondary Goals:
- Calculate the average subscription duration and compare by subscription type and region.
- Track and visualize the churn rate, particularly focusing on early cancellations (within 6 months).
- Provide segmented data insights by region and subscription type.

- Added Calculated Columns:
  i) Duration: Calculated in months for each subscription.

### 6. Insights and Observations
- Churn Rate: We observed that the majority of cancellations occur within the first 6 months, particularly in the [East Region].
  
- Subscription Duration: The average subscription duration is highest for the [Premium Subscription, annual plans], suggesting stronger customer retention for longer-term subscriptions.
  
- Revenue by Region: [The Southern Region] contributes the most to total revenue, driven by [The Basic Subscription Type].

### 7. Recommendations
Based on the insights gained, the following actions are recommended:

- Targeted Retention Campaigns: Implement campaigns for new customers within the first 6 months to reduce early cancellations.
  
- Promote Long-Term Plans: Offering incentives for longer-term plans could boost average subscription duration.
  
- Focus on High-Performing Regions: Further marketing efforts in high-revenue regions to maximize returns.

### 8. Conclusions. 
This Excel-based project provides a detailed look into customer subscription patterns and serves as a foundation for more advanced analysis. The structured data insights and recommendations will assist stakeholders in making strategic decisions to enhance customer retention and revenue growth.
